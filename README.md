# SmartHydrate - Water Intake Tracker

A simple and clean web application to help you track your daily water intake and maintain healthy hydration habits.

## Features

- **Home Page**: Attractive landing page with hydration information and benefits
- **Weekly Water Tracker**: Track your water intake for each day of the week (8 glasses per day goal)
- **Interactive Checkboxes**: Click on bars to mark glasses as consumed (green checkmarks)
- **Age-Based Recommendations**: View recommended daily water intake based on age groups
- **Responsive Design**: Works on desktop and mobile devices
- **Progress Saving**: Save your daily water tracking progress
- **Active Navigation**: Highlighted navigation showing which page you're on

## How to Use

1. **Open the Application**:
   - Open `water.html` in your web browser to start

2. **Navigate**:
   - Click "Home" to view hydration information
   - Click "Tracker" to track your daily water intake
   - Click "Start Tracking" button on home page to go directly to tracker

3. **Track Your Water**:
   - Enter your name and age in the input fields
   - Click on the glass bars to mark them as consumed (they turn green)
   - Each day has 8 glasses to track
   - Click "Save Progress" when done

4. **View Recommendations**:
   - Scroll down to see recommended water intake by age group

## Technologies Used

## Setup Instructions

1. Download all three files (`water.html`, `tracker.html`, `water.css`)
2. Keep them in the same folder
3. Open `water.html` in any modern web browser
4. No server or installation required
